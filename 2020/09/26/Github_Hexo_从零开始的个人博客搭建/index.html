    git是目前最先进的分布式版本控制系统,他能够跟踪所有文本文件的内容改动和文件大小变化，比如txt文件、md文件、网页、程序代码等。同时他也能跟踪图片、音频、视频等文件的改动，但由于这些文件本质属于二进制编码，因此具体改了什么地方，git是不知道的。
